## 💡 使用技巧

### 技巧 1: 组合使用

```css
/* 从模板 1 选择消息气泡样式 */
.message-user .message-content-container {
  background: linear-gradient(...);
}

/* 从模板 4 选择玻璃效果 */
.message-content-container {
  backdrop-filter: blur(10px);
}

/* 添加你自己的创意 */
.markdown h1 {
  color: var(--color-primary);
  text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.2);
}
```

---

### 技巧 2: 渐进式增强

```
第一步: 应用基础模板
      ↓
第二步: 调整颜色和字体
      ↓
第三步: 添加特效和动画
      ↓
第四步: 优化细节
```

---

### 技巧 3: 主题切换

保存多个版本的 CSS 代码:

```
📁 我的主题
```

需要切换时，复制对应文件内容即可！
