## 🎯 开始之前

### 你需要知道的三件事

```
┌─────────────────────────────────────────┐
│  1. 选择器 (Selector)                    │
│     告诉 CSS "要改变什么"                 │
│     例如: .message-user                  │
│                                          │
│  2. 属性 (Property)                      │
│     告诉 CSS "要改变的具体内容"           │
│     例如: background-color               │
│                                          │
│  3. 值 (Value)                          │
│     告诉 CSS "改成什么样"                 │
│     例如: #ff0000                        │
└─────────────────────────────────────────┘

完整示例:
.message-user {           ← 选择器
  background: #ff0000;    ← 属性: 值
}
```

### 选择器使用示例

```css
/* 🎯 精确定位：只改变用户消息的背景 */
.message-user .message-content-container {
  background: #ff6b6b;
}

/* 🎯 精确定位：只改变 AI 消息的背景 */
.message-assistant .message-content-container {
  background: #4ecdc4;
}

/* 🎯 全局改变：所有 Markdown 内容 */
.markdown {
  font-size: 16px;
}

/* 🎯 更精确：只改变 Markdown 中的标题 */
.markdown h1,
.markdown h2,
.markdown h3 {
  color: #ff6b6b;
}
```

### 颜色使用示例

```css
/* ✅ 推荐：使用 CSS 变量 */
.my-element {
  background: var(--color-primary);
  color: var(--color-text);
  border: 1px solid var(--color-border);
}

/* ❌ 不推荐：写死颜色值 */
.my-element {
  background: #00b96b;
  color: white;
  border: 1px solid #ccc;
}
```

```
│  └────┴──────────────────────────────────────┘ │
└─────────────────────────────────────────────────┘

┌──────────────────────────────────────┐
│  平板 (768px - 1024px)                │
├──────────────────────────────────────┤
│  ┌──────────────────────────────────┐│
│  │  中等边距                         ││
│  │  中等字体                         ││
│  └──────────────────────────────────┘│
└──────────────────────────────────────┘

┌────────────────────────┐
│  移动端 (< 768px)       │
├────────────────────────┤
│  ┌────────────────────┐│

│  │  较小字体           ││
│  └────────────────────┘│
└────────────────────────┘
```

**CSS 代码:**
```css
/* 平板适配 */
@media (max-width: 1024px) {
  .message-content-container {
    max-width: 90% !important;
  }
}

/* 移动端适配 */
@media (max-width: 768px) {
  .message-content-container {
    max-width: 95% !important;
    padding: 10px 12px !important;
  }
  
  .markdown {
    font-size: 14px !important;
  }
}
```
